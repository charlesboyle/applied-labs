How it works

  1. 1
    Install SigInspector Add it from the Chrome Web Store. No account, no setup.
  2. 2
    Open DevTools Press F12 or ⌘⌥I on any page using Statsig.
  3. 3
    Click the SigInspector tab Find it in the DevTools panel bar alongside Elements and Console.
  4. 4
    Navigate and capture Events appear live as the SDK fires them. Click any row for full JSON detail.
